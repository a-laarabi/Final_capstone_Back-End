# 📖 car Rentals  <a name="about-project"></a>

Our application is an App for a car rental agencies primarily serve people who requires a temporary vehicle, It allows users to create an account, login, and book a car for a specific period of time.

## Backend

The frontend was made with React and can be found [here](https://github.com/a-laarabi/Final_capstone_Front-End)
The backend was made with Ruby on Rails and can be found [here](https://github.com/a-laarabi/Final_capstone_Back-End)

## 💻 Getting Started <a name="getting-started"></a>

To get a local copy up and running, follow these steps.

### Prerequisites

In order to run this project you need:

```sh
 node 
 yarn
```

### Setup

Clone this repository to your desired folder:

```sh
  cd my-folder
  git clone https://github.com/a-laarabi/Final_capstone_Back-End.git
  git clone https://github.com/a-laarabi/Final_capstone_Front-End.git
```

### Install

Install this project with:

```sh
  cd Final_capstone_Back-End
  npm install
```

### Usage

To run the project, execute the following command:

```sh
  npm start
```
